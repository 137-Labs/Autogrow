# 🌿 AutoGrow

A minimalist, privacy-first, offline-ready progressive web application designed for home horticultural tracking and grow room metrics management. 

[![Live App](https://img.shields.io/badge/Live%20Demo-Online-brightgreen?style=for-the-badge&logo=github)](https://137-labs.github.io/Autogrow/)
[![Platform](https://img.shields.io/badge/Platform-Mobile%20%2F%20Desktop-black?style=for-the-badge)](https://137-labs.github.io/Autogrow/)
[![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)](LICENSE)

---

## 🚀 Features

- **📱 Single-File Architecture:** Fully self-contained inside a high-performance single `index.html` payload. No complex databases, server installs, or backend modules required.
- **🔐 Total Data Sovereignty:** 100% client-side data architecture. Your seed bank inventory, feeding schedules, and logs remain completely inside your browser's local memory—never uploaded to an external cloud or third-party server.
- **🌱 Integrated Utilities:** Built-in modular panels optimized for tracking cycles, atmospheric changes, seed vault catalogs, and standard data logging workflows.
- **🔋 Offline-First Engine:** Once opened, the application fully caches locally. Access your data, logs, and tracking metrics deep in indoor grow tents or offline locations without losing connection capability.
- **🕶️ Minimalist UI:** Styled with a responsive, high-contrast, all-black dark mode interface designed for quick navigation and optimal nighttime viewing in dark environments.

## 🛠️ How to Use & Install

### Option 1: Run the Live Demo (Easiest)
Simply tap the link below in any mobile or desktop web browser to launch your personal application dashboard immediately:
👉 **[Launch AutoGrow Live](https://137-labs.github.io/Autogrow/)**

### Option 2: Run Locally (Desktop or Mobile)
1. Download the `index.html` asset from this repository.
2. Double-click the file to open it natively in your browser of choice (Chrome, Safari, Firefox, Samsung Internet).
3. The app is immediately functional and fully running on your local device storage.

### Option 3: Install as an App (PWA)
Because AutoGrow is built as a **Progressive Web App (PWA)**, you can save it directly onto your phone or computer home screen:
- **Android / Samsung:** Open the live link in your browser, tap the browser menu (three vertical dots), and select **"Add to Home screen"** or **"Install app"**.
- **iOS:** Open the live link in Safari, tap the **Share** icon at the bottom center, scroll down, and select **"Add to Home Screen"**.

## 🔒 Security & Optimization Note

To prevent plagiarism, reverse-engineering, and unauthorized asset redistribution, the production pipeline for this application utilizes cryptographic script compilation. 

- Core runtime algorithms, array processing architectures, and data structures are securely transformed into unreadable, mangled hexadecimal fragments using an offline **JavaScript Obfuscation engine**.
- This layer guarantees full browser compilation speeds and zero execution lag, while preventing structural scraping or intellectual property theft.
